# TryDelta Frontend Mock Beta

Versao isolada de preview com frontend funcional e API mock local em memoria.

## Escopo

- Sem dependencia de banco de dados
- Sem dependencia de API externa
- Login, dashboard, clientes e acoes principais com dados simulados

## Execucao local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## E2E

```bash
npm run test:e2e
```
